## General Concept

 - Virtual Machines 
	 - Host - OS that runs the VM software
	 - Guest - VM software that runs the OS
- Networking 
	- NAT 
		- Port Forwarding (punch network to the guest vm network)
	- Private ( Multiple vm to communicate with each other)

### Tools

 - Virtual Box 
	 - Virtualbox.org
	 - Oracle
	 - Guest Additions
 - Git 
	 - Scripts
	 - Configuration
	 - Documentation
 - Vagrant 
 - Git Bash (Windows)
### What is vagrant
 - Manage VM software (VirtualBox)
	 - Create/Destroy VMs
	 - Start/Stop/restart VMs
	 - Access to VMs
	 - Networking/ VM settings
- Orchestrates "provisioning"
	- On demand setup
- Shared Folders

### Two Architecture

 - Kitchen Sink (All in one)
	 - Host runs VMs only
	 - IDE/Editor in VM
	 - GUI in VM
	 - All work in VM
	 - Deploy in VM
 - Light Server
	 - Host has IDE/Editor
	 - Host runs VM
	 - Shared folder
	 - VM for local deploy
	 - No GUI for VM
