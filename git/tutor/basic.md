## Basic command Overview

### Starting with a fresh project

    $ pwd
    /User/bjohn
    $ mkdir projects
    $ cd  projects
    $ git init fresh-project
    $ cd fresh-project
    $ ls -al
    $ cd .git
    $ ls
    $ cd ..
    $ git status
    <Analyse>
    $ vim hipster.txt
    <write one paragraph>
    $ git status
    $ git add hipster.txt
    $ git status
    $ git commit
    <Invoke the default editor>
    $ git status

> Note: root-commit means very first commit.
> Copt text from [hipster website](http://hipsum.co)
